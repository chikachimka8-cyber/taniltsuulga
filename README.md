# Танилцуулга вэб сайт

Энэхүү жижиг статик вэб сайт нь таны товч танилцуулга (портфолио/профайл) зориулсан загвар юм.

Файлууд:
- index.html — Нүүр хуудас
- styles.css — Стилиуд
- script.js — Жижиг интерактив үйлдлүүд

Хэрхэн ажиллуулах:

1) Хялбар шалгах (нь Python 3 шаардлагатай):

```bash
python -m http.server 8000
```

Дараа нь шүүгээг нээгээд http://localhost:8000 руу очно.

2) Файлуудыг өөрчлөх:
- `index.html` дахь нэр, танилцуулга, чадвар, имэйл-г өөрчилнө.
- Хэрэгтэй бол зураг `images/` хавтасанд байрлуулаад `index.html`-д заана.

3) Дараагийн алхмууд (хэрвээ хүсвэл би туслахаар):
- GitHub Pages дээр байрлуулаад, эсвэл Vercel/Netlify-д деплой хийх.
- Контакт формыг сервер талд илгээх API-тай холбох.

Хүсвэл би ижил загварыг Git репозиторт оруулж, GitHub Pages рүү холбоод өгч болно.
